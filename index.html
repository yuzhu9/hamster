仓鼠

仓鼠，又名地鼠，是仓鼠亚科（学名：Cricetinae）动物的通称。共七属十八种，主要分布于亚洲，少数分布于欧洲，其中中国有三属八种。目前，仓鼠多被当作家庭的宠物饲养，另外也因为易于人工繁殖，经常被动物实验使用。

在野外，仓鼠是黄昏时出没的动物，白天的时间多半会待在地底下以避免被捕食。他们主要的食物来源是种子、水果以及蔬菜，偶尔也会吃穴居昆虫。而除分布在中亚的小仓鼠外，其他种类的仓鼠两颊皆有颊囊，从臼齿侧延伸到肩部。可以用来临时储存或搬运食物回洞储藏，故名仓鼠。

历史

虽然早在1839年，乔治·罗伯特·瓦特豪斯就首度合乎科学地描述了叙利亚仓鼠（黄金鼠）的特称，但是直到1939年，研究人员才能成功地繁殖并驯养仓鼠。当时，整个实验室的仓鼠似乎是由单一对兄妹/姐弟仓鼠繁殖出的后代，这窝仓鼠是在1930年，耶路撒冷大学的动物学家以色列·阿哈罗尼抓获，并将它们从叙利亚的阿勒坡进口而来，而后成功地在耶路撒冷繁殖。

多年以后，这个繁殖物种被出口到美国，并在此变成最受欢迎的宠物和实验动物之一。在比较研究中，驯养的仓鼠有比较少的遗传变异，然而，在行为学、时间生物学、形态计量学、血液学的差异上则是落在物种间变异的预期之内。

特征

外貌
仓鼠有分布于两颊的小胡须，虽似老鼠，但尾略短，藏匿于皮毛之中，难以见着，但是有中国品种的仓鼠，尾长达2～3cm。四肢短小，前肢承担起摄食的主要辅助作用。背部皮毛常有条纹状，随仓鼠的成长，条纹色也会加深，当然也有没有条纹品种的仓鼠。面颊内有囊，可以储食，因其存在而可见得面颊鼓起。上颚会有对锐利的门牙以啃噬。 成年公仓鼠体重在85-130g之中，而其母仓鼠为95-150g。在欧洲仓鼠可有体重达250g-600g。仓鼠寿命在18-24月之间，也有特别例子，能养到两年以上。
感官
仓鼠视力不佳，它们都是近视并且色盲的，只能分辨黑白。而在侧腹或是下腹部有香腺，让仓鼠在挖掘基材或垫材时可以留下气味。仓鼠也利用嗅觉辨别食物以及彼此的性别。它们对高频率的噪音特别敏感，也可以听见超音波并以此沟通。
食性
仓鼠是杂食性的动物，虽然宠物仓鼠可以吃鼠粮，但是也可以喂蔬菜、水果、和坚果类的食物。有些生活在中东地区的仓鼠会捕捉昆虫为食。仓鼠都是后肠发酵，会食用自己的粪便，重新获取那些消化过但是没有被吸收的养分。
繁殖
成年后的公鼠于2周后能够交配，而母鼠则在6-10周后能够交配。母仓鼠发情周期为4天。产后不再发情，以及生产的胎数为2-22只。

行为

虽然它时常清理毛发，但仍然难以消除身上一种异味，所以人若养之，必须每天以鼠沙辟臭。

仓鼠有一对不断生长的门牙，三对臼齿，齿型为：1 0 0 3，成交错排列的三棱体。臼齿具齿根，或不具齿根而终生能生长。

该科各种类动物基本都属中小型鼠类。体长在5－28厘米之间，体重在30－200克。体型短粗。尾短，一般不超过身长的四分之一，部分品种不超过后腿长度的一半，甚至根本看不到。主要食物为植物种子，喜食坚果，亦食植物嫩茎或叶，偶尔也吃小虫。多数不冬眠，冬天靠储存食物生活。少数品种在天气寒冷情况下会进入不太活跃的准冬眠状态。

仓鼠主要在夜间活动，视力差，只能模糊辨形，颜色只能分辨黑白。嗅觉灵敏。毛色繁杂。全年皆可繁殖，每胎5－12只。平均寿命2－3年。

仓鼠部分品种因为和人类亲近，所以就成为近年流行的宠物，如黄金仓鼠、加卡利亚仓鼠、坎贝尔仓鼠、罗伯罗夫仓鼠等。

分类

仓鼠科共七属十八种，可分为：

  • 短耳仓鼠属
  • 甘肃仓鼠属
  • 仓鼠属
  • 欧洲仓鼠：黑腹仓鼠
  • 中仓鼠属：叙利亚仓鼠，又称黄金鼠
  • 侏儒仓鼠：为坊间多数宠物鼠的种类
    • 坎贝尔侏儒仓鼠：俗称一线鼠
    • 小毛足鼠：俗称老公公鼠
    • 短尾侏儒仓鼠：俗称三线鼠
